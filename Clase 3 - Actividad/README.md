# Clase 3 - Actividad

## Consigna resuelta

- Réplica de una noticia actual utilizando etiquetas semánticas de HTML.
- Inclusión de navegación con enlaces (`nav` + `a`).
- Inclusión de formulario de suscripción con:
  - `form`
  - `label`
  - `input` (text, email, radio, checkbox)
  - `button`

## Archivos

- `index.html`: resolución principal.
- `serve.py`: servidor estático local.

## Ejecución local

```bash
python3 serve.py
```

Luego abrir: `http://127.0.0.1:8081/index.html`
